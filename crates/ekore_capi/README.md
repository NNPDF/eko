# ekore_capi

C API for the [ekore](https://crates.io/crates/ekore) crate, part of the [EKO](https://github.com/NNPDF/eko) framework.

This crate exposes the anomalous dimensions and operator matrix elements from `ekore` through a stable C ABI, making them callable from C, C++, or any language with a C FFI. The C header is generated by [cbindgen](https://github.com/mozilla/cbindgen) via [`cargo-c`](https://crates.io/crates/cargo-c).
